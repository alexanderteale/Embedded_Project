Instalation
======

###From GIT

```sh
$ git clone https://github.com/alexanderteale/Embedded_Project
$ cd ./Embedded_Project
$ ./bootstrap.sh

```

###From Tar
```sh
$ wget www.adress.of.tar.gz
$ tar -xvf Embedded_Project.tar.gz
$ cd ./Embedded_Project
$ ./bootsrap.sh
```

#####Optional Install to Environment

```sh
$ sudo make install
```
